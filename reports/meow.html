Meow

Machine Very Easy Linux Telnet 2025-06-28

Summary

Meow is the first machine most users will experience on HTB. It is a very simple Linux mchine with and exposed Telnet service with deafualt credntials.

Reconnaissance

Nmap

nmap -sV 0.0.0.0

Key ports discovered:

  • 23/tcp ‚ÄĒ Telnet

Exploitation

Exposed Service with Default Credentials

Telnet was exposed on port 23 allowing for any connection from attackers. The first credentials attempted were "root" with no password which allowed root access to the service.
